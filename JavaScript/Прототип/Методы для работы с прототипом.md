#### 1. **`Object.getPrototypeOf(obj)`**
Получить прототип объекта.

```js
const obj = {};
console.log(Object.getPrototypeOf(obj) === Object.prototype); // true
```

#### 2. **`Object.setPrototypeOf(obj, proto)`**
Установить прототип объекта (медленно, лучше использовать `Object.create`).

```js
const parent = { name: 'Parent' };
const child = {};
Object.setPrototypeOf(child, parent);
console.log(child.name); // 'Parent'
```

#### 3. **`Object.create(proto)`**
Создать объект с указанным прототипом.

```js
const proto = { greet: () => 'Hello' };
const obj = Object.create(proto);
console.log(obj.greet()); // 'Hello'
```

#### 4. **`obj.hasOwnProperty(prop)`**
Проверить, есть ли свойство у самого объекта (не в прототипе).

```js
const obj = Object.create({ inherited: 'from proto' });
obj.own = 'own property';

console.log(obj.hasOwnProperty('own'));      // true
console.log(obj.hasOwnProperty('inherited')); // false
```

#### 5. **`prop in obj`**
Проверить наличие свойства (включая прототип).

```js
const obj = Object.create({ inherited: 'value' });
obj.own = 'value';

console.log('own' in obj);       // true
console.log('inherited' in obj); // true
console.log('missing' in obj);   // false
```

#### 6. **`obj instanceof Constructor`**
Проверить, был ли объект создан через указанный конструктор.

```js
function User() {}
const user = new User();

console.log(user instanceof User);        // true
console.log(user instanceof Object);     // true (через цепочку)
```

#### 7. **`Object.getOwnPropertyNames(obj)`**
Получить все собственные свойства объекта (включая неперечисляемые).

```js
const obj = { a: 1, b: 2 };
console.log(Object.getOwnPropertyNames(obj)); // ['a', 'b']
```

#### 8. **`Object.keys(obj)`**
Получить перечисляемые собственные свойства объекта.

```js
const obj = { a: 1, b: 2 };
console.log(Object.keys(obj)); // ['a', 'b']
```
