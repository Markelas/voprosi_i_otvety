**Атрибуты — это дополнительные параметры тега, которые задают свойства элемента.**

|**Тип атрибутов**|**Примеры**|**Комментарий**|
|---|---|---|
|**Глобальные**|`id`, `class`, `style`, `title`, `hidden`, `tabindex`, `data-*`|Работают с любым тегом|
|**Специфические**|`<a>` → `href`, `target`;  <br>`<img>` → `src`, `alt`, `width`, `height`;  <br>`<input>` → `type`, `value`, `placeholder`, `checked`, `disabled`;  <br>`<form>` → `action`, `method`|Применимы только к конкретным тегам|
