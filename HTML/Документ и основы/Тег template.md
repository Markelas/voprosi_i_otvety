# Тег `<template>`

## Краткий ответ для собеседования

**`<template>`** — HTML5-элемент для хранения разметки, которая **не отображается** и **не выполняется** при загрузке страницы. Содержимое **инертное**: картинки не грузятся, скрипты не запускаются, стили не применяются к странице. Разметка хранится в **DocumentFragment** (свойство `content`). Нужен, чтобы хранить шаблоны и клонировать их в DOM по требованию через JavaScript (`content.cloneNode(true)`). Удобен для повторяющихся блоков (списки, карточки, модалки). На собеседовании: что это, зачем (инертность, шаблоны), как использовать (clone + append).

---

## Что такое `<template>`

**`<template>`** — семантический контейнер для **шаблонной разметки**, которую браузер не показывает и не обрабатывает при загрузке страницы.

- Содержимое **не отображается** в документе.
- Содержимое **инертное**: изображения не загружаются, `<script>` не выполняется, `<style>` внутри template не применяется к странице (не «вытекает» наружу).
- Разметка хранится в **DocumentFragment** — свойство **`content`** у элемента `<template>`. Её клонируют и вставляют в DOM когда нужно.

Простыми словами: это «заготовка» HTML в документе, которая не участвует в отрисовке и загрузке ресурсов, пока вы не вставите её в DOM через JS.

---

## Зачем нужен

- **Хранить разметку без немедленного отображения** — в отличие от скрытого `<div>` (display: none) картинки и скрипты внутри `<template>` не грузятся и не выполняются до вставки.
- **Шаблоны для повторяющихся блоков** — один раз описать разметку карточки/строки, клонировать по данным и вставлять в DOM.
- **Валидная разметка** — внутри можно писать фрагменты (например, только `<tr>`), которые невалидны в том месте документа, где лежит template (например, не внутри `<table>`).

---

## Как использовать

- Разметка доступна как **`templateElement.content`** (DocumentFragment).
- **Клонировать:** `templateElement.content.cloneNode(true)`.
- **Вставить** клон в нужное место: например, `container.append(clone)`.
- После вставки клонированная разметка ведёт себя как обычный HTML: грузятся ресурсы, выполняются скрипты, применяются стили.

## Кратко запомнить

**`<template>`** — тег для хранения шаблонной разметки с **инертным** содержимым в **DocumentFragment** (`content`). Не отображается, не грузит ресурсы, не выполняет скрипты до вставки. Использование: клонировать `content.cloneNode(true)` и вставить в DOM — удобно для повторяющихся блоков и динамического контента без фреймворка.
