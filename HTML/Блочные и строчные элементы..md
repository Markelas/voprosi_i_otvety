| **Признак**                  | **Блочные (block)**                                 | **Строчные (inline)**                        |
| ---------------------------- | --------------------------------------------------- | -------------------------------------------- |
| Занимают всю ширину родителя | Да                                                  | Нет                                          |
| Перенос на новую строку      | Да                                                  | Нет                                          |
| Можно задавать ширину/высоту | Да                                                  | Нет (обычно игнорируется)                    |
| Примеры                      | `<div>`, `<p>`, `<h1>`, `<section>`, `<ul>`, `<li>` | `<span>`, `<a>`, `<strong>`, `<em>`, `<img>` |
