## Data-* атрибуты

**Data-* атрибуты** — пользовательские атрибуты, которые позволяют хранить дополнительную информацию внутри HTML-элементов и передавать её в JavaScript через свойство **dataset** без нарушения валидности HTML.

**Для чего нужны:** хранение данных в элементах (ID, конфигурация, метаданные), передача из HTML в JS, валидный HTML, удобный доступ через `element.dataset`.

**Синтаксис:** начинаются с `data-`, имя в нижнем регистре с дефисами, значение — строка. Пример: `data-user-id="123"`.

**Доступ в JS:** через **dataset** — дефисы превращаются в camelCase: `data-user-id` → `element.dataset.userId`. Альтернатива: `getAttribute('data-user-id')` / `setAttribute()`.

**Важно:** значения в dataset всегда строки; для числа или boolean нужна конвертация (parseInt, `=== 'true'`).

**Когда использовать:** ID элементов, конфигурация компонентов, метаданные для JS, делегирование событий. Не для визуального стиля (CSS), не для доступности (ARIA), не для SEO (семантика).
