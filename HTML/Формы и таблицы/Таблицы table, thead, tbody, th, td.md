# Таблицы: \<table\>, \<thead\>, \<tbody\>, \<th\>, \<td\>

## Краткий ответ для собеседования

**\<table\>** — семантический контейнер для табличных данных (строка/столбец). **\<thead\>** — блок заголовков колонок, **\<tbody\>** — тело таблицы, **\<tfoot\>** — итоги (опционально). **\<tr\>** — строка, **\<th\>** — ячейка-заголовок, **\<td\>** — ячейка с данными. **\<caption\>** — подпись к таблице. На собеседовании могут спросить: когда уместна таблица (табличные данные, не верстка), зачем thead/tbody, разницу между **th** и **td** (th — заголовок; для доступности скринридеры используют th для навигации).

---

## Когда использовать таблицу

**Таблица** в HTML предназначена для **табличных данных**: строки и столбцы с логической связью (расписание, сравнение, отчёты). Не для верстки страницы (для сеток используют flex/grid). Использование table для layout устарело и плохо для доступности и адаптива.

---

## Структура

- **\<table\>** — обёртка.
- **\<caption\>** — подпись к таблице (опционально).
- **\<thead\>** — секция заголовков (строки с **th**).
- **\<tbody\>** — основное содержимое (строки **tr** с **td**).
- **\<tfoot\>** — секция «итого» (опционально).

Порядок в разметке: thead, tbody, tfoot. **tr** только внутри thead/tbody/tfoot; **th** и **td** только внутри **tr**.

---

## th и td

- **\<th\>** — ячейка-заголовок (заголовок столбца или строки). Атрибут **scope**: col / row — для доступности.
- **\<td\>** — обычная ячейка с данными.

**colspan** и **rowspan** у th и td — объединение ячеек.

---

## Доступность

- **caption** или заголовок перед таблицей — чтобы было понятно, о чём таблица.
- **th** с **scope** — скринридер связывает ячейки с заголовками.

---

## Вопросы на собеседовании

| Вопрос | Краткий ответ |
|--------|----------------|
| Когда уместно использовать table? | Для табличных данных (строки/столбцы с логической связью), не для верстки страницы. |
| Зачем thead и tbody? | Семантика (заголовки отдельно от данных), удобно стилизовать и делать скролл только tbody. |
| В чём разница между th и td? | th — ячейка-заголовок (столбца/строки), td — данные; th важен для доступности и стилей. |
| Что такое scope у th? | Указывает ось заголовка: col (столбец) или row (строка); помогает скринридерам. |
| Для чего caption? | Подпись к таблице, поясняет содержание; полезна для доступности и SEO. |

---

## Кратко для ответа

**table** — для табличных данных. **thead** — заголовки, **tbody** — данные, **tfoot** — итоги. **th** — заголовок ячейки (scope для доступности), **td** — данные. **caption** — подпись. Таблицу не используют для верстки; для сеток — flex/grid.
