# Flexbox

## Краткий ответ для собеседования

**Flexbox** (Flexible Box Layout) — модель раскладки для **одномерного** размещения элементов в строку или столбец. Родитель получает **display: flex**, дочерние элементы становятся flex-элементами и выстраиваются по главной оси (row/column). Основные свойства контейнера: **flex-direction** (направление оси), **justify-content** (выравнивание по главной оси), **align-items** (по поперечной оси), **flex-wrap** (перенос на новую строку). У элементов: **flex-grow**, **flex-shrink**, **flex-basis** (или кратко **flex**) — как делить свободное место и сжиматься. На собеседовании часто спрашивают: что такое flex, чем главная и поперечная ось отличаются, зачем justify-content и align-items, что делает flex: 1.

---

## Что такое Flexbox

**Flexbox** — это способ раскладки: контейнер с **display: flex** (или **inline-flex**) управляет расположением своих прямых потомков (flex-элементов) вдоль **главной оси** (по умолчанию — горизонталь, слева направо) или **поперечной оси** (перпендикулярно). Элементы можно выравнивать, распределять свободное место и переносить на новую строку.

---

## Контейнер (родитель)

| Свойство | Описание |
|----------|----------|
| **flex-direction** | row (по умолчанию), row-reverse, column, column-reverse — направление главной оси. |
| **justify-content** | Выравнивание по **главной** оси: flex-start, flex-end, center, space-between, space-around, space-evenly. |
| **align-items** | Выравнивание по **поперечной** оси: stretch, flex-start, flex-end, center, baseline. |
| **flex-wrap** | nowrap (по умолчанию), wrap, wrap-reverse — перенос элементов на новую строку/столбец. |
| **gap** | Зазор между flex-элементами (в современных браузерах). |

**Главная ось** — та, вдоль которой идут элементы (задаётся flex-direction). **Поперечная ось** — перпендикулярна ей. justify-content действует по главной, align-items — по поперечной.

---

## Элементы (дети)

| Свойство | Описание |
|----------|----------|
| **flex-grow** | Насколько элемент может вырасти, чтобы занять свободное место (число, по умолчанию 0). |
| **flex-shrink** | Насколько может сжаться при нехватке места (число, по умолчанию 1). |
| **flex-basis** | Базовый размер до распределения места (например, 200px или auto). |
| **flex** | Краткая запись: flex-grow flex-shrink flex-basis. **flex: 1** часто значит «занять доступное место» (1 1 0% или 1 1 auto в зависимости от контекста). |
| **align-self** | Переопределить align-items для одного элемента. |

**flex: 1** — типичный способ «растянуть» элемент на всё свободное пространство в строке (grow = 1, shrink = 1, basis = 0 или auto).

---

## Когда использовать

Flexbox удобен для: выравнивания по центру (justify-content + align-items), навигационных полос, карточек в ряд, распределения места между блоками, простых адаптивных сеток с переносом (flex-wrap). Для сложных двумерных сеток чаще используют **Grid**.

## Вопросы на собеседовании

| Вопрос | Краткий ответ |
|--------|----------------|
| Что такое Flexbox? | Модель раскладки: контейнер с display: flex выстраивает детей по главной оси; управление через justify-content, align-items, flex-wrap и свойства элементов (flex-grow, flex-shrink). |
| Чем главная и поперечная ось отличаются? | Главная — направление строки/столбца (flex-direction); поперечная — перпендикулярно. justify-content — по главной, align-items — по поперечной. |
| Что делает flex: 1? | Элемент растёт (grow) и сжимается (shrink) с базой 0 или auto — занимает доступное место в контейнере. |
| Как центрировать блок по центру с Flexbox? | У контейнера display: flex; justify-content: center; align-items: center; (для одного блока по центру). |
| Flexbox или Grid? | Flexbox — одномерная раскладка (строка или столбец). Grid — двумерная (строки и столбцы одновременно). Для простых рядов и центровки — Flexbox; для сеток — Grid. |

---

## Кратко запомнить

**Flexbox** — display: flex на контейнере; элементы выстраиваются по главной оси. **justify-content** — по главной оси, **align-items** — по поперечной. **flex: 1** — занять доступное место. Одномерная раскладка; для двумерных сеток — Grid.
