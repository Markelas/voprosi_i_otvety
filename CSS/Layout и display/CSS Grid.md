# CSS Grid

## Краткий ответ для собеседования

**CSS Grid** — модель раскладки для **двумерных** сеток: строки и столбцы задаются у контейнера (**display: grid**), дочерние элементы размещаются в ячейках. Контейнер: **grid-template-columns** / **grid-template-rows** (размеры колонок и строк), **gap** (зазоры), **justify-items**, **align-items**. Элементы могут задавать своё положение: **grid-column**, **grid-row** или **grid-area**. На собеседовании часто спрашивают: чем Grid отличается от Flexbox, что такое grid-template-columns, зачем gap, как растянуть элемент на несколько ячеек.

---

## Что такое Grid

**Grid** — раскладка по сетке: контейнер с **display: grid** определяет **столбцы** и **строки**, а прямые потомки (grid-элементы) автоматически попадают в ячейки или занимают несколько ячеек. Подходит для карточек, дашбордов, сложных макетов страницы (header, sidebar, content, footer).

---

## Контейнер (родитель)

| Свойство | Описание |
|----------|----------|
| **grid-template-columns** | Размеры столбцов (например, 1fr 1fr 1fr, 200px 1fr, repeat(3, 1fr)). |
| **grid-template-rows** | Размеры строк. |
| **gap** (grid-gap) | Зазор между строками и столбцами (row-gap, column-gap). |
| **justify-items** / **align-items** | Выравнивание содержимого ячеек по горизонтали и вертикали. |
| **place-items** | Краткая запись: align-items justify-items. **place-items: center** — центр по обоим осям. |

**fr** — единица доли свободного пространства (1fr 2fr — второе в два раза шире первого).

---

## Элементы (дети)

| Свойство | Описание |
|----------|----------|
| **grid-column** | Начало и конец по столбцам (например, 1 / 3 или span 2). |
| **grid-row** | Начало и конец по строкам. |
| **grid-area** | Имя области или краткая запись row-start / col-start / row-end / col-end. |

Элемент по умолчанию занимает одну ячейку; через grid-column/grid-row можно растянуть на несколько.

---

## Grid vs Flexbox

- **Flexbox** — **одномерная** раскладка: одна строка или один столбец, перенос (wrap) даёт следующую строку, но явного управления строками и столбцами нет.
- **Grid** — **двумерная**: явно задаётся сетка строк и столбцов, элемент может занимать любую область. Для сеток, макетов страницы, выравнивания по двум осям — Grid; для рядов кнопок, навбара, центровки одного блока — часто Flexbox.

## Вопросы на собеседовании

| Вопрос | Краткий ответ |
|--------|----------------|
| Что такое CSS Grid? | Модель раскладки: контейнер с display: grid задаёт строки и столбцы; дети размещаются в ячейках, можно растягивать на несколько (grid-column, grid-row). |
| Чем Grid отличается от Flexbox? | Grid — двумерная сетка (строки и столбцы). Flexbox — одномерная (одна ось, перенос). |
| Что такое fr в Grid? | Единица доли свободного места: 1fr 2fr — второе в два раза больше первого. |
| Как задать три равные колонки? | grid-template-columns: repeat(3, 1fr); |
| Как растянуть элемент на несколько ячеек? | grid-column: 1 / 3; (или span 2), grid-row при необходимости. |

---

## Кратко запомнить

**Grid** — display: grid; задаются **grid-template-columns** и **grid-template-rows**; **gap** — зазоры. Элементы размещаются в ячейках; **grid-column** / **grid-row** — занять несколько ячеек. **fr** — доля пространства. Двумерная раскладка; для одномерных рядов — Flexbox.
