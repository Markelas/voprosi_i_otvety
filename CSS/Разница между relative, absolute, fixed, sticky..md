· <span class="blue">**relative**</span> — элемент смещается относительно своего обычного положения, при этом занимает место на странице.

· <span class="red">**absolute**</span> — элемент позиционируется относительно ближайшего позиционированного предка, снимается из нормального потока, не занимает места.

· <span class="purple">**fixed**</span> — элемент фиксируется относительно окна браузера и остаётся на месте при прокрутке.

· <span class="green">**sticky**</span> — ведёт себя как `relative`, пока не достигнет заданной позиции при прокрутке, после чего становится фиксированным.
