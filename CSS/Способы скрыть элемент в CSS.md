<span class="red">**display: none**</span> - Полностью удаляет элемент из потока документа (из layout). Элемент **не занимает место**, **не виден**, **недоступен**. DOM-узел остаётся в дереве, но визуально как будто его нет.

<span class="orange">**visibility: hidden**</span> **-** Элемент **остаётся в потоке**, **занимает место**, но **не виден**. Может быть использован для **плавного появления** через visibility: visible.

<span class="purple">**opacity: 0**</span> **-** Делает элемент **полностью прозрачным**, но он **остается в DOM и в layout**. В отличие от visibility: hidden, элемент **можно кликнуть** (он всё ещё интерактивен).

<span class="green">**width: 0; height: 0** **-**</span> Делает элемент **не видимым**, но **не удаляет из layout**. Часто используется для **скрытия технических элементов** (например, скрытые input). Может вызывать **переполнение** или другие визуальные проблемы.
