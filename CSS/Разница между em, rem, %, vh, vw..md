·<span class="red"> % </span> — размер относительно родительского элемента (например, ширина 50% — половина ширины родителя).

· <span class="green">vh</span> и <span class="blue">vw</span> — единицы высоты и ширины окна браузера (1vh = 1% высоты окна).

· <span class="blue">em</span>  — размер относительно шрифта родительского элемента (если у родителя 20px, то 2em = 40px).

· <span class="pink">rem</span>  — размер относительно шрифта корневого элемента (**_html_**), удобно для глобального масштабирования.  

<u class="blue">**_Относительные_**</u>: em — относительно размера шрифта родителя, масштабируется вместе с ним. % — относительно размеров родительского элемента (например, ширина 50%).
<u class="red">**_Абсолютные_**</u>: например, px — фиксированная единица, не зависит от других элементов.
