<span class="blue">**🌳 1. Потомки (пробел)**</span>

` div p { color: red; }`

<span class="">Выбирает ВСЕ `p` внутри `div` (на любом уровне)</span>

<span class="orange">**👆 2. Прямые потомки (>)**</span>


`div > p { color: blue; }`

<span class="">Выбирает ТОЛЬКО прямых потомков `p` внутри `div`</span>

<span class="purple">**🎯 3. Множественные классы (без пробела)**</span>

`.class1.class2 { color: green; }`

<span class="">Элемент с ОБОИМИ классами: `class="class1 class2"`</span>

<span class="red">**➕ 4. Соседние элементы (+)**</span>

`h1 + p { color: orange; } `

<span class="">Первый `p` сразу после `h1`</span>

<span class="yellow">**🌊 5. Общие соседи (~)**</span>

`h1 ~ p { color: pink; }`

<span class="">ВСЕ `p` после `h1` (на том же уровне)</span>

---

<span class="red">**💡 Запомни: пробел = все потомки, > = прямые потомки, .class.class = оба класса**</span>
