## ⏱️ Базовый промис - delay(ms)

### Задача

Напишите функцию `delay(ms)`, которая возвращает промис. Этот промис должен резолвиться через указанное количество миллисекунд.

---

### Зачем это нужно?

Понять, как создавать промисы с нуля. Используется для искусственных задержек в тестах, анимациях.

---

### Решение:

```js
function delay(ms) {
    return new Promise((resolve) => {
        setTimeout(() => {
            resolve();
        }, ms);
    });
}
```

---

### Пример использования:

```js
delay(1000).then(() => console.log('Выполнено через 1 секунду'));

// С async/await
async function example() {
    await delay(2000);
    console.log('Выполнено через 2 секунды');
}
```

---

### Объяснение:

1. Создаём новый `Promise` с функцией-исполнителем
2. Используем `setTimeout` для задержки
3. Вызываем `resolve()` после истечения времени
4. Промис резолвится через указанное количество миллисекунд

---

### Вариант с возвратом значения:

```js
function delay(ms, value) {
    return new Promise((resolve) => {
        setTimeout(() => {
            resolve(value);
        }, ms);
    });
}

// Использование
delay(1000, 'Готово!').then(result => console.log(result)); // 'Готово!'
```
