## ➕ Сумма двух чисел в отсортированном массиве

### Задача

Дан отсортированный массив и target. Найти индексы двух чисел, сумма которых равна target.

---

### Решение:

```js
function twoSumSorted(nums, target) {
  let left = 0, right = nums.length - 1;

  while (left < right) {
    let sum = nums[left] + nums[right];

    if (sum === target) return [left, right];
    if (sum < target) left++;
    else right--;
  }

  return [-1, -1];
}
```

---

### Объяснение:

- Два указателя: один слева, один справа
- Если сумма меньше target → увеличиваем сумму (двигаем `left`)
- Если сумма больше target → уменьшаем сумму (двигаем `right`)
- Если равна → возвращаем индексы

---

### ⚠️ Отличие от задачи "Две суммы" (unsorted):

Для **несортированного массива** используй **HashMap**:

```js
function twoSum(nums, target) {
  const map = new Map();

  for (let i = 0; i < nums.length; i++) {
    if (map.has(target - nums[i])) {
      return [map.get(target - nums[i]), i];
    }
    map.set(nums[i], i);
  }
}
```
