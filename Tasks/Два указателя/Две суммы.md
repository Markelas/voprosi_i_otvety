## ➕ Две суммы (Two Sum)

### Задача

Дан массив целых чисел `nums` и число `target`. Нужно вернуть индексы двух чисел, сумма которых равна `target`.

Можно считать, что есть ровно одно решение, и одно и то же число нельзя использовать дважды.

---

### Пример:

```js
sumTwoDigit([9, 7, 11, 15, 2], 9) // вернет [1,4]
```

---

### Решение:

```js
function sumTwoDigit(nums, target) {
  const map = new Map();

  for (let i = 0; i < nums.length; i++) {
    if (map.has(target - nums[i])) {
      return [map.get(target - nums[i]), i];
    }
    map.set(nums[i], i);
  }
}
```

---

### Объяснение:

- **`target - nums[i]`** — какое число нам нужно, чтобы в сумме с `nums[i]` получилось `target`
- **`map.has(...)`** — проверяем, видели ли мы уже это число ранее
- Если нашли → возвращаем индексы: `map.get(target - nums[i])` (найденное) и `i` (текущее)
