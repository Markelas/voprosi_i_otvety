[[Каррирование]] - это процесс, при котором функция с несколькими аргументами  
преобразуется в **цепочку функций**, каждая из которых принимает **по одному аргументу**.

```js
function curry(func) {
  return function curried(...args) {
	// если аргументов достаточно — выполняем функцию
    if (args.length >= func.length) {
      return func(...args);
    } else {
    // иначе возвращаем новую функцию, которая ждёт оставшиеся аргументы
      return (...nextArgs) => curried(...args, ...nextArgs);
    }
  };
}
```

<font color="#31859b">Механизм:</font>
- Каждое частичное применение (каждый вызов) возвращает **новую функцию**.
- Эта функция копит аргументы и **рекурсивно вызывает** `curried(...)`.
- В <font color="#31859b">func(</font><font color="#e36c09">...args</font><font color="#31859b">)</font> складываются аргументы, если вдруг сделали один вызов с несколькими аргументами, например, <font color="#31859b">curriedSum(</font><font color="#e36c09">10, 20, 30</font><font color="#31859b">)</font>
- Как только аргументов ≥ `fn.length` — выполняется `fn`.

<font color="#e36c09">Пример работы:</font>

```js
// Здесь указано определенное количество
function sum(a, b, c) {
  return a + b + c;
}

// Далее, вызываем созданную функцию и она рекурсивно считает
const curriedSum = curry(sum);

// 1. Вызов по частям:

console.log('По частям (1)(2)(3):', curriedSum(1)(2)(3)); // Ожидаемый вывод: 6

// 2. Вызов с промежуточным сохранением:

const addFive = curriedSum(5);

console.log('Промежуточный (5)(6)(7):', addFive(6)(7)); // Ожидаемый вывод: 18

// 3. Вызов сразу:

console.log('Сразу (10, 20, 30):', curriedSum(10, 20, 30)); // Ожидаемый вывод: 60

```

<font color="#e36c09">Что делает:</font>

- Знает, **сколько аргументов ожидает** оригинальная функция (`fn.length`).
- Собирает их частями, **пока не наберёт нужное количество**.
- Когда достаточно — **выполняет `fn` и возвращает результат**.
